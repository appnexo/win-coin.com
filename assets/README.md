# Assets Folder

Coloca aquí las imágenes de tu sitio web:

## Imágenes Recomendadas

### 1. og-image.png
- **Dimensiones:** 1200x630px
- **Uso:** Imagen de vista previa cuando compartes el sitio en redes sociales
- **Contenido sugerido:** Logo + texto "Gana Dinero Real" + captura de app

### 2. logo.png
- **Dimensiones:** 512x512px (transparente)
- **Uso:** Logo principal de Win Coin
- **Formato:** PNG con fondo transparente

### 3. app-icon.png
- **Dimensiones:** 512x512px
- **Uso:** Ícono de la aplicación
- **Formato:** PNG (mismo que usas en Google Play)

### 4. Capturas de Pantalla (Opcional)
- screenshot1.png - Pantalla principal
- screenshot2.png - Pantalla de anuncios
- screenshot3.png - Pantalla de retiros
- **Dimensiones:** Resolución de tu dispositivo de prueba
- **Uso:** Galería de imágenes en el sitio (si decides agregarla)

### 5. google-play-badge.png (Opcional)
Ya se usa el badge oficial de Google desde CDN, pero puedes descargarlo localmente si prefieres.

## Optimización de Imágenes

Para mejor rendimiento:
- Comprime las imágenes antes de subirlas
- Usa formatos WebP si es posible (fallback a PNG/JPG)
- Mantén los tamaños de archivo bajo 500KB

## Herramientas Recomendadas

- **TinyPNG:** https://tinypng.com/ (compresión)
- **Canva:** https://www.canva.com/ (diseño de og-image)
- **Photopea:** https://www.photopea.com/ (editor gratuito tipo Photoshop)
